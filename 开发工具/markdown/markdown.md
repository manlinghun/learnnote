# 无序列表

使用 -、+ 和 * 表示无序列表，前后留一行空白，可嵌套，例如

- 1
  - 1.1
    - 1.1.1
  - 1.2
- 2

+ 1
  + 1.1
    + 1.1.1
  + 1.2
+ 2

* 1
  * 1.1
    * 1.1.1
  * 1.2
* 2

# 有序列表

使用 1. （点号后面有个空格）表示有序列表，可嵌套。

1. 1
   1. 1.1
      1. 1.1.1
   2. 1.2
2. 2

# 流程图

主要的语法为 name=>type: describe，其中 type 主要有以下几种：
1. 开始和结束：start end
2. 输入输出：inputoutput
3. 操作：operation
4. 条件：condition
5. 子程序：subroutine

```flow
st=>start: 开始
op=>operation: 输入一个数字
cond=>condition: 大于10
e=>end
st->op->cond
cond(yes)->e
cond(no)->op
```
